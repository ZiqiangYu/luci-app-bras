    About

    luci-app-bras是一个东南大学Bras的OpenWRT路由器客户端，主要包括了xl2tpd的配置模板、东南大学校内路由表和Web控制界面。

    利用该工具，你可以使用浏览器控制Bras，并且通过设置防火墙转发规则，通过WLAN接入的设备能共享Bras。

    Dependency

    • xl2tpd

    Installation

    在Releases下载安装包。

    # 1. 安装 luci-app-bras
$ opkg install luci-app-bras_0.1-1_all.ipk

# 2. 添加防火墙转发规则，参见firewall.user
