About

luci-app-bras 是东南大学 Bras 的 OpenWRT 客户端。

Dependency

  • xl2tpd

Installation

在 Releases 下载安装包

# 1. 安装 luci-app-bras
$ opkg install luci-app-bras_0.1-1_all.ipk

# 2. 添加防火墙转发规则，参见firewall.user
