# luci-app-bras

`luci-app-bras`是适用于`OpenWrt`的 BRAS 服务的 Web 前端，目前仅支持东南大学 BRAS。

## Dependency

* xl2tpd

## Installation
    $ wget https://github.com/xuchunyang/luci-app-bras/releases/download/v0.9-alpha/luci-app-bras_0.1-1_all.ipk
    $ opkg luci-app-bras_0.1-1_all.ipk
